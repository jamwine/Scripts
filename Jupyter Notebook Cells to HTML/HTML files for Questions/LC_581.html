581. Shortest Unsorted Continuous Subarray

Given an integer array nums, you need to find one continuous subarray that if you only sort this subarray in ascending order, then the whole array will be sorted in ascending order.

Return the shortest such subarray and output its length.

Example 1:
Input: nums = [2,6,4,8,10,9,15]
Output: 5
Explanation: You need to sort [6, 4, 8, 10, 9] in ascending order to make the whole array sorted in ascending order.

Example 2:
Input: nums = [1,2,3,4]
Output: 0

Example 3:
Input: nums = [1]
Output: 0

Constraints:
1 <= nums.length <= 104
-105 <= nums[i] <= 105

Follow up: Can you solve it in O(n) time complexity?

Solution

class Solution:
    def findUnsortedSubarray(self, array: List[int]) -> int:
        if len(array) < 2:
            return 0
        minimum = float('inf')
        maximum = float('-inf')
        for i in range(len(array)):
            if self.unsorted(i, array):
                minimum = min(minimum, array[i])
                maximum = max(maximum, array[i])
        if minimum == float('inf'):
            return 0

        left_idx = 0
        while minimum >= array[left_idx]:
            left_idx += 1

        right_idx = len(array)-1
        while maximum <= array[right_idx]:
            right_idx -= 1

        return right_idx-left_idx+1

    def unsorted(self, i, array):
        if i == 0:
            return array[i] > array [i+1]
        if i == len(array)-1:
            return array[i] < array[i-1]
        return array[i-1] > array[i] or array[i] > array[i+1]

Time complexity : O(N), where N are the total elements in an array

Space complexity : O(1)
