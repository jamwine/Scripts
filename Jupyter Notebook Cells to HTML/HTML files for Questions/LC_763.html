763. Partition Labels

A string S of lowercase English letters is given. We want to partition this string into as many parts as possible so that each letter appears in at most one part, and return a list of integers representing the size of these parts.

Example 1:

Input: S = "ababcbacadefegdehijhklij"
Output: [9,7,8]

Explanation:
The partition is "ababcbaca", "defegde", "hijhklij".
This is a partition so that each letter appears in at most one part.
A partition like "ababcbacadefegde", "hijhklij" is incorrect, because it splits S into less parts.

Note:

S will have length in range [1, 500].
S will consist of lowercase English letters ('a' to 'z') only.

Solution

class Solution:
    def partitionLabels(self, S: str) -> List[int]:
        partitionLengths = []

        #Keep track of last occurrences of each letter (dictionary)
        lastIndexes = {char: i for i, char in enumerate(S)}

        max_ch_idx, partition = 0, 0

        for i, char in enumerate(S):
            max_ch_idx  = max(max_ch_idx, lastIndexes[char])
            if i == max_ch_idx:
                partitionLengths.append(i + 1 - partition)
                partition = i + 1 # (i+1) will be the next position

        return partitionLengths

Time complexity : O(n), n is the length of the string

Space complexity : O(1), dictionary of fixed length (not more than 26 characters)
