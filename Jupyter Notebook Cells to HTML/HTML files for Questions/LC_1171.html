1171. Remove Zero Sum Consecutive Nodes from Linked List

Given the head of a linked list, we repeatedly delete consecutive sequences of nodes that sum to 0 until there are no such sequences.

After doing so, return the head of the final linked list.  You may return any such answer.

Example 1:

Input: head = [1,2,-3,3,1]
Output: [3,1]
Note: The answer [1,2,1] would also be accepted.

Example 2: 

Input: head = [1,2,3,-3,4]
Output: [1,2,4]

Solution

# Definition for singly-linked list.
# class ListNode:
#     def __init__(self, val=0, next=None):
#         self.val = val
#         self.next = next

class Solution:
    def removeZeroSumSublists(self, l1: ListNode) -> ListNode:

        head = ListNode(0)
        head.next = l1
        prefix = 0
        d = {prefix:head} # {prefix sum : last node of getting this sum value}

        while l1:
            prefix += l1.val
            d[prefix] = l1
            l1 = l1.next

        # Second Iteration
        l1 = head
        prefix = 0
        while l1:
            prefix += l1.val
            l1.next = d[prefix].next
            l1 = l1.next

        return head.next

Time complexity : O(n), two pass algorithm, n is the number of nodes in the linked list.

Space complexity : O(n), for maintaining a dictionary
