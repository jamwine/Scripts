1360. Number of Days Between Two Dates

Write a program to count the number of days between two dates.

The two dates are given as strings, their format is YYYY-MM-DD as shown in the examples.


Example 1:
Input: date1 = "2019-06-29", date2 = "2019-06-30"
Output: 1


Example 2:
Input: date1 = "2020-01-15", date2 = "2019-12-31"
Output: 15


Constraints:
The given dates are valid dates between the years 1971 and 2100.

Solution

class Solution:        
    month_days = {1: 31, 2: 28, 3: 31, 4: 30, 5: 31,
                  6: 30, 7: 31, 8: 31, 9: 30, 10: 31, 11: 30, 12: 31}

    def is_leap(self, year):
        if year % 400 == 0: return True
        if year % 100 == 0: return False        
        return year % 4 == 0

    def move_day(self, y, m, d):        
        d += 1
        month_last_day = self.month_days[m]
        if m == 2 and self.is_leap(y): month_last_day += 1
        if d > month_last_day:
            d = 1
            m += 1

        if m == 13:
            m = 1
            y += 1   

        return y, m, d

    def daysBetweenDates(self, date1: str, date2: str) -> int:
        if date1 > date2: date1, date2 = date2, date1
        y1, m1, d1 = list(map(int, date1.split('-')))
        y2, m2, d2 = list(map(int, date2.split('-')))
        days = 0

        while y1 < y2 or m1 < m2 or d1 < d2:
            y1, m1, d1 = self.move_day(y1, m1, d1)
            days += 1
        return days

Time complexity : O(date2-date1)

Space complexity : O(1)
