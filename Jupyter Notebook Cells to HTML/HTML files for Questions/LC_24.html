24. Swap Nodes in Pairs

Given a linked list, swap every two adjacent nodes and return its head.
You may not modify the values in the list's nodes, only nodes itself may be changed.

Example 1:


Given 1->2->3->4, you should return the list as 2->1->4->3.

Solution

# Definition for singly-linked list.
# class ListNode:
#     def __init__(self, val=0, next=None):
#         self.val = val
#         self.next = next

class Solution:
    def swapPairs(self, l1: ListNode) -> ListNode:
        head = ListNode(0)
        p = head
        head.next = slow = fast = l1

        while True:

            count = 0
            while fast and count < 2:   # fast locates the range
                fast = fast.next
                count += 1

            if count == 2:  # if size k satisfied, reverse the inner linked list
                prev_node=fast
                curr_node=slow
                for _ in range(2):
                    next_node = curr_node.next
                    curr_node.next = prev_node
                    prev_node = curr_node
                    curr_node = next_node

                # connect two k-groups
                p.next =  prev_node
                p = slow
                slow = fast

            else:
                return head.next

Time complexity : O(n)

Space complexity : O(1)
