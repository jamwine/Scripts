498. Diagonal Traverse

Given an m x n matrix, return an array of all the elements of the array in a diagonal order.

Example 1:
Input: mat = [[1,2,3],[4,5,6],[7,8,9]]
Output: [1,2,4,7,5,3,6,8,9]


Example 2:
Input: mat = [[1,2],[3,4]]
Output: [1,2,3,4]

Constraints:
m == mat.length
n == mat[i].length
1 <= m, n <= 104
1 <= m * n <= 104
-105 <= mat[i][j] <= 105

Solution

class Solution:
    def findDiagonalOrder(self, matrix: List[List[int]]) -> List[int]:
        height, width = len(matrix)-1, len(matrix[0])-1
        row, col = 0, 0
        result = []

        while not (row < 0 or row > height or col < 0 or col > width):
            result.append(matrix[row][col])
            if (row + col) % 2 == 0:
                if col == width:
                    row += 1
                elif row == 0:
                    col += 1
                else:
                    row -= 1
                    col += 1
            else:
                if row == height:
                    col += 1
                elif col == 0:
                    row += 1
                else:
                    row += 1
                    col -= 1

        return result

Time complexity : O(N), where N is the total number of elements in matrix.

Space complexity : O(N)
