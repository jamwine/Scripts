567. Permutation in String

Given two strings s1 and s2, return true if s2 contains a permutation of s1, or false otherwise.

In other words, return true if one of s1's permutations is the substring of s2.


Example 1:
Input: s1 = "ab", s2 = "eidbaooo"
Output: true
Explanation: s2 contains one permutation of s1 ("ba").


Example 2:
Input: s1 = "ab", s2 = "eidboaoo"
Output: false


Constraints:
1 <= s1.length, s2.length <= 10^4
s1 and s2 consist of lowercase English letters.

Solution

class Solution:
    def checkInclusion(self, s1: str, s2: str) -> bool:

        target = [0] * 26
        seen = [0] * 26

        for ch in s1:
            ch_idx = ord(ch)-ord('a')
            target[ch_idx] += 1

        left, right = 0, 0
        n = len(s2)

        while right < n: 
            # decrease the window size, remove the left character
            if right - left == len(s1):
                ch_idx = ord(s2[left])-ord('a')
                seen[ch_idx] -= 1
                left += 1

            # increase the window size, add the right character
            ch_idx = ord(s2[right])-ord('a')
            seen[ch_idx] += 1
            right += 1

            # if current string matches the target string, add the left index in result
            if seen == target:
                return True

        return False

Time complexity : O(N), N is the length of bigger string

Space complexity : O(1)
