4. Median of Two Sorted Arrays

Given two sorted arrays nums1 and nums2 of size m and n respectively, return the median of the two sorted arrays.

Follow up: The overall run time complexity should be O(log (m+n)).

Example 1:
Input: nums1 = [1,3], nums2 = [2]
Output: 2.00000
Explanation: merged array = [1,2,3] and median is 2.

Example 2:
Input: nums1 = [1,2], nums2 = [3,4]
Output: 2.50000
Explanation: merged array = [1,2,3,4] and median is (2 + 3) / 2 = 2.5.

Example 3:
Input: nums1 = [0,0], nums2 = [0,0]
Output: 0.00000

Example 4:
Input: nums1 = [], nums2 = [1]
Output: 1.00000

Example 5:
Input: nums1 = [2], nums2 = []
Output: 2.00000


Constraints:
nums1.length == m
nums2.length == n
0 <= m <= 1000
0 <= n <= 1000
1 <= m + n <= 2000
-106 <= nums1[i], nums2[i] <= 106

Solution

class Solution:
    def findMedianSortedArrays(self, nums1: List[int], nums2: List[int]) -> float:
        len1, len2 = len(nums1), len(nums2)

        if len1 > len2:
            return self.findMedianSortedArrays(nums2, nums1)

        start, end = 0, len1
        while start <= end:
            partitionx = (start + end) // 2
            partitiony = (len1 + len2 + 1)//2 - partitionx

            maxLeftX = float(-inf) if (partitionx == 0) else nums1[partitionx - 1]
            maxLeftY = float(-inf) if (partitiony == 0) else nums2[partitiony - 1]
            minRightX = float(inf) if (partitionx == len(nums1)) else nums1[partitionx]
            minRightY = float(inf) if (partitiony == len(nums2)) else nums2[partitiony] 

            #median should be around the four elements that satisfies the below condition
            if (maxLeftX <= minRightY) and (maxLeftY <= minRightX): 
                if (len1 + len2) % 2 == 0:
                    median = (max(maxLeftX, maxLeftY) + min(minRightX, minRightY))/2
                    return median
                else:
                    median = max(maxLeftX, maxLeftY)
                    return median

            #shift left and right pointers accordingly    
            elif maxLeftY > minRightX:
                start = partitionx + 1
            else:
                end = partitionx - 1
        return median

Time complexity : O(log(min(m,n))), where m and n are the two arrays.

Space complexity : O(1)
