926. Flip String to Monotone Increasing

A binary string is monotone increasing if it consists of some number of 0's (possibly none), followed by some number of 1's (also possibly none).

You are given a binary string s. You can flip s[i] changing it from 0 to 1 or from 1 to 0.
Return the minimum number of flips to make s monotone increasing.

Example 1:
Input: s = "00110"
Output: 1
Explanation: We flip the last digit to get 00111.

Example 2:
Input: s = "010110"
Output: 2
Explanation: We flip to get 011111, or alternatively 000111.

Example 3:
Input: s = "00011000"
Output: 2
Explanation: We flip to get 00000000.

Constraints:
1 <= s.length <= 105
s[i] is either '0' or '1'.

Solution

class Solution:
    def minFlipsMonoIncr(self, s: str) -> int:
        zeros, ones = 0,0
        for i in s:
            if i == '0':zeros+=1

        # check zeros on the right side and ones on the left side of the pivot.
        # For 11011: initially there is 1 zero on right side and 0 ones on the left side: so (zeros,ones) = (1,0)
        # Next, temp = [(1,0),(1,1),(1,2),(0,2),(0,3),(0,4)]
        # so: min(zeros+ones)= min(1, 2, 3, 2, 3, 4) = 1

        flips = zeros
        for i in s:
            if i =='0':zeros-=1
            else:ones+=1
            flips=min(flips, zeros + ones)
        return flips

Time complexity : O(N)

Space complexity : O(1)
