92. Reverse Linked List II

Reverse a linked list from position m to n. Do it in one-pass.
Note: 1 ≤ m ≤ n ≤ length of list.

Example 1:


Input: 1->2->3->4->5->NULL, m = 2, n = 4
Output: 1->4->3->2->5->NULL

Solution

# Definition for singly-linked list.
# class ListNode:
#     def __init__(self, val=0, next=None):
#         self.val = val
#         self.next = next
class Solution:        

    def reverseBetween(self, l1: ListNode, m: int, n: int) -> ListNode:
        if not l1:
            return None

        head = ListNode(0)
        p = head
        head.next = l1 

        # p is the pointer at position m-1
        for _ in range(m-1):
            p=p.next

        # curr_node points to position m
        curr_node=p.next 
        prev_node = None

        for _ in range(n - m + 1):
            next_node = curr_node.next
            curr_node.next = prev_node
            prev_node = curr_node
            curr_node = next_node

        # Making connections
        p.next.next = curr_node # Connecting 2->5 via 1
        p.next = prev_node # Connecting 1->4

        return head.next

Time complexity : O(n), the algorithm makes one traversal of the list of n nodes.

Space complexity : O(1) since we simply adjust some pointers in the original linked list
