42. Trapping Rain Water

Given n non-negative integers representing an elevation map where the width of each bar is 1, compute how much water it is able to trap after raining.

Example 1:


Input: [0,1,0,2,1,0,1,3,2,1,2,1]
Output: 6

Solution

class Solution:
    def trap(self, height: List[int]) -> int:
        if len(height) == 0:
            return 0

        totalWater = 0
        leftMax = []

        # each index has a left and right building, will be 0 in case of first index
        leftMax.append(0)

        # leftMax represents the unrealistic total height of elevation (L -> R)
        for i in range(len(height)-1):
            leftMax.append(max(leftMax[i], height[i]))

        rightMax = 0
        # rightMax represents the unrealistic total height of elevation (R -> L)
        for i in reversed(range(len(height))):
            rightMax = max(rightMax, height[i])

            # The gap represents the height filled with water: min(leftMax, rightMax)
            gap = min(leftMax[i], rightMax)

            # Difference between gap and actual height provides the stored water
            if gap > height[i]:
                totalWater += gap - height[i] 

        return totalWater

Time complexity : O(n)

Space complexity : O(n)
